# Vanishing_Point
本ツールは、画像から**消失点（Vanishing Point）を自動検出**するC++アプリケーションです。  
**OpenCVを用いてエッジ抽出および*ハフ変換**による直線検出を行い、それらの交点の重み付けにより最も確からしい消失点を推定します。

## 1 機能概要
・OpenCVを用いた エッジ検出（Canny）
・ハフ変換による 直線検出
・交点の重み付けによる 消失点の自動推定
・直線延長描画による視覚的な結果確認
・結果のウィンドウ表示（交点・直線オーバーレイ

## 2 検出アルゴリズムの概要
  1. 入力画像をグレースケール変換
  2. Canny法でエッジ検出
  3. ハフ変換（HoughLinesP）で線分を抽出
  4. 線分同士の交点を計算
  5. 角度や位置関係に基づき交点に重み付け
  6. 最も重みの大きい交点を「消失点」として選定

## 3 実行環境
  ・ Language: C++17 以降  
  ・ Library: OpenCV 4.12  
  ・ OS: Windows11  

## 4 既知の課題
ハフ変換パラメータ（閾値や最小線分長）が静的設定であるため、入力画像の情報量によっては線分の見地に課題があります。  
  1. 線分の検知過多（ノイズが多い画像）
  2. 線分の検知不足（淡いエッジの画像）

## 5 今後の改善案
  1. 画像の明度やエッジ密度に応じた 動的パラメータ調整
  2. RANSAC等を用いた 外れ値除去
  3. 機械学習による 複数消失点の自動推定
